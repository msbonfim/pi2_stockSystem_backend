Teste de Acessibilidade do Django Admin

Se você está vendo este arquivo, os arquivos estáticos estão sendo servidos corretamente.

Instruções para Testar:

  1. Acesse: http://127.0.0.1:8000/admin/
  2. Abra o Console do Navegador (F12)
  3. Procure por mensagens relacionadas a "Accessibility"
  4. Procure por um botão flutuante no canto inferior direito com o símbolo ♿
  5. Verifique se os arquivos CSS estão carregando na aba "Network"
